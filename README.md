# THE MET landing
This is a page of the MET museum created using a given design.  
You can find here exhibitions schedule, museum contacts and work hours.
## Tech stack
 - HTML
 - SCSS
 - Flexbox
 - Grid
 - BEM
 - Moibile and tablet adaptation
